## 1. Что такое view в DB2 for z/OS

- **Обычное представление (VIEW)** — в каталоге хранится только текст `SELECT …`; при обращении к view Db2 читает данные **с базовых таблиц**.
- **«Хранимая вьюшка» в смысле данных на диске** — это не классический VIEW, а **MQT** или таблица-слепок с `REFRESH`/`LOAD`.

Конкатенацию во view можно вынести; **ускорения без индексов часто нет** — план может совпасть с исходным.
